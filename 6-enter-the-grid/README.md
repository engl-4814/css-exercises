# Enter the Grid with Template Columns &amp; Rows

Practice grid layout work by designing a grid layout that defines a template and also uses:

1. Line-placement technique with **grid areas**
2. Both flexible and absolute values
3. Both `repeat()` and `minmax()`

Your grid layout doesn't have to be super flexible yet. If it looks great on a laptop-size screen, you're good for now. We will spend some time working on flexibility soon!

For now, focus on creating a grid that you find in the wilds of the world. Wireframe it. Snap it a an image, and include it in your project folder.

## Folder Setup

Follow this procedure to setup your coding folder:

1. Within this root folder, create a new folder with your lastname.
   - **NOTE** `lastname` is your last name. Mine would be: `lindgren`.
2. Inside the root of *your personal coding folder*, create the an HTMl file named `index.html`.
3. Again, inside the root of *your personal coding folder*, create an `assets` folder.
4. Inside `assets`, create a `.keep` file, add your wireframe image, and create a `css` folder.
5. Inside `css`, create `grids.css` to link to `index.html`.
6. `※\(^o^)/※` Start writing some code! `※\(^o^)/※`
