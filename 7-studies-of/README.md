# Exercise 7 - Studies of ...

## Goal

Practice using CSS flexbox and grid by being inspired by Simmons' playful 'studies' from her [LayoutLand lab](https://labs.jensimmons.com/#flexibility). Do so by creating 2 studies that focus on a particular design strategy that we've been reading about, such as contrast and similarity, figure-ground, proximity, or from Simmons' page with overlap, flexibility, etc.

At least one of the studies should use grid-areas from Powell's video.

You have total reign over the type of study and the artifact that you choose to create.

**Please include wireframes** that show me how you planned out your grid work. You can save them as image files within the root of your folder.

## Folder Setup

Follow this procedure to setup your coding folder:

1. Within this root folder, create a new folder with your lastname.
   - **NOTE** `lastname` is your last name. Mine would be: `lindgren`.
2. Inside the root of *your personal coding folder*, create the 2 HTMl files named by the particular focus of your study. For example, a study of overlap with grid and flexbox could be named `overlap.html`.
3. Again, inside the root of *your personal coding folder*, create an `assets` folder.
4. Inside `assets`, create a `.keep` file and a `css` folder.
5. Inside `css`, create `style.css` to link to your `.html` files.
6. `※\(^o^)/※` Start writing some code! `※\(^o^)/※`
