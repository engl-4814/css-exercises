# 1- CSS Selection & Box Model Practice

**IMPORTANT** For this particular assignment, please do not push your code iteratively as usual. Please wait until just before class. That way, everyone has an opportunity to complete the assignment on their own.

1. Copy the `lindgren.html` file. Make sure it is within the root of this `1-selection-box-model` folder.
2. Rename this copied `.html` file with your last name.
3. Copy the `style.css` file in the `assets/css` folder. Make sure this copy is within the `assets/css` folder.
4. Rename this copied `.css` file with your last name.
5. Follow the prompts in the `.html` and `.css` documents to complete the assignment. 
    - **NOTE**: There is only one prompt in the `.html` file to change the path to your specific CSS file. If you do not change this path, your html file will not change based on your CSS within your particular file.

## Tips

- Open your html and css files in the vertical pane view, so you can see both throughout the assignment.
- Practice selecting one element at a time, adding a ruleset, and then testing it in your browser **before moving onto the next prompt**.
- As noted above, use your browser's inspector tool after each selection and ruleset. If you check them one at a time, you can ensure that you catch any errors and problems.
